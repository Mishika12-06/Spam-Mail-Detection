# SMS Spam Detection using Machine Learning
This project builds a machine learning model to classify SMS messages as Spam or Ham (Not Spam).
# Problem Statement
Spam messages are a common issue in digital communication. This project aims to automatically detect spam messages using NLP and classification algorithms.
# Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- TF-IDF Vectorizer
# Workflow
1. Data Cleaning
2. Text Preprocessing
3. Feature Extraction (TF-IDF)
4. Model Training
5. Model Evaluation
# Model Used
 Logistic Regression 
# Result
Achieved accuracy of 95.07%.
